## 🌱 关于这个仓库

- 在这里，你将会看到我知识体系的构建过程，主要包括以下几个部分：

  * **🧠 技术笔记**: 系统性地整理编程语言、框架、算法、操作系统等硬核知识。

  * **✍️ 零散实践**: 偶然间学到的一些个人认为很有用的东西，会在这里进行上传。

  * **💡 心得与反思**: 超越代码本身，沉淀关于学习方法、职业规划、问题解决思路的思考。

  * **📔 个人日记**: 记录日常的学习进度、遇到的 Bug 和一些灵光一闪的想法。
